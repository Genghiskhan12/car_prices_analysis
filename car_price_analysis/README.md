# Анализ цен на автомобили

**Цель проекта:** Определить ключевые факторы, влияющие на цену автомобиля (бренд, пробег, год выпуска).

**Данные:** 1200 записей о продажах автомобилей (поля: make, model, price, mileage, year, mmr).
Перед анализом из данных были исключены строки с некорректными значениями бренда (0 или null). Остальные пропуски в ключевых полях (цена, пробег, год) отсутствуют.

**Инструменты:** SQL (MySQL), GitHub.

Вопросы:

1.Средняя цена по маркам — make, sellingprice

2.Топ-5 самых дорогих машин — make, model, sellingprice

3.Зависимость цены от пробега 

4.Средняя цена по годам — year, sellingprice

5.Разница между ценой продажи и оценкой — sellingprice - mmr как premium

6.Разброс цен по маркам — MAX(sellingprice) - MIN(sellingprice) на каждую марку

7.Количество машин по годам — year, COUNT(*)
